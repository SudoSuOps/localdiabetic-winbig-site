Why I built this

I didn’t build LocalDiabetic to start another tech company. I built it because I’m living this.

I’m a Type 1 diabetic. I’ve lost toes. I’ve dealt with infections. And I learned diabetes isn’t just about your blood sugar — it’s about everything between appointments. The meals, the walks, the medications, the foot checks. The little things that become big problems if you miss them.

There wasn’t one simple place that helped me manage my day while keeping my data private. So I started building it — a calm LifeBoard, and a Bee that works for me, not for an advertising company.

Small daily wins can prevent tomorrow’s emergencies.

If it helps save one foot, one hospitalization — or helps one person catch a problem a little earlier — then every late night building it has been worth it.

Donovan
Type 1 diabetic · not just building LocalDiabetic — building the tool I wish I had
